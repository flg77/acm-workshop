    Create Hosted Control Plane

    ** Hosted Control Planes** is a technology preview feature, it shouldn’t be used in production clusters. In this section, we will enable the feature and deploy a Hosted Cluster on AWS. On the Clusters page, select once again Create Cluster. Select Amazon Web Services and Hosted mode. You’ll face the official instructions on how to create a hosted cluster. 1 - Turn on Hosted Control Planes functionality The hosted control planes feature is available as a technology preview item and it’s disabled by default.
